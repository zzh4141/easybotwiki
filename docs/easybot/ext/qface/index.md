## 安装

下载好后,将资源解压至 `plugins/ItemsAdder/contents`

![](./image/install.png)

:::info 关于重载
EasyBot支持IA重载资源包,你可以直接使用`iazip`命令重载资源包,EasyBot检测到`QFace`会自动启用IA表情功能。

![](./image/ok.png)
:::
